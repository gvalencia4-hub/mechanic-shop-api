# Mechanic Shop API

This project is a REST API built with Flask using the Application Factory Pattern.

It simulates a mechanic shop system where customers create service tickets and mechanics can be assigned or removed from those tickets.

## Technologies Used
- Python
- Flask
- SQLAlchemy
- Marshmallow
- SQLite

## Project Features
- Create customers
- Create mechanics
- Create service tickets
- Assign mechanics to tickets
- Remove mechanics from tickets
- View service tickets
