# Defeat the Evil Wizard (Advanced Python OOP)

Turn-based battle game where you choose a hero class and fight the Evil Wizard.

## Features

- 4 character classes: Warrior, Mage, Archer, Paladin
- 2 unique abilities per class
- Randomized attack damage
- Healing system (health cannot exceed max)
- Evil Wizard regenerates health and attacks each turn
- Victory / Defeat message at the end

## How to Run

```bash
python wizard_game.py
```
